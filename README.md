# Break-Even Analysis System

A simple Python project that calculates Break-Even Point and basic business metrics.

---

## Features

- Calculate Total Cost
- Calculate Total Revenue
- Calculate Profit
- Calculate Break-Even Point (Units & Revenue)
- Simple business insight (profit or loss)
- Graph visualization using Matplotlib
- Interactive terminal input

---

## Technologies

- Python 3
- Matplotlib

python breakevenpoint.py
